# Story: Implement quick scan path (default baseline)

## Context

The existing 6-file baseline must remain the default `quick` mode. This story formalizes that path behind the new CLI.

## Acceptance criteria

- **AC-E01-4:** Running `wize-dev-kit document-project` (or `document-project quick`) produces exactly the 6 existing baseline files:
  - `overview.md`
  - `architecture-snapshot.md`
  - `conventions.md`
  - `dependencies.md`
  - `risk-spots.md`
  - `open-questions.md`
- Existing files are preserved and updated (`last_refreshed` bumped).
- Behavior is equivalent to the current `/wize-document-project` skill.

## Out of scope

- New docs like `index.md`, `source-tree-analysis.md` (other modes).
